The Company Profile Portal is a full-stack web application designed to streamline internal employee onboarding and data management.Technologies Used
The project uses a standard Full-Stack Architecture split into three main layers:
1. Frontend (The User Interface)
   HTML5
   CSS3
   JavaScript
2. Backend (The Application Server)
   Python
   Flask
3. Database (The Storage Layer)
   SQLite
   Flask-SQLAclchemy
